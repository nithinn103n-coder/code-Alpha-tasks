# CodeAlpha Data Analyst Tasks

Welcome to my **CodeAlpha** internship project repository! This repository contains a complete data analysis and visualization portfolio focused on extracting, processing, and understanding data from a curated dataset of the top 100 global companies. 

The pipeline is split into distinct structured phases—moving from core statistical data profiling to advanced visual insight generation using Python's scientific stack.
